### 概念
1. Metric Space(X,d)
   
   X is a set
   
   d is a function d:X × X -> R , 并且有如下三个属性:

   a. 非负性 
   $
   \quad x,y\in X \quad, d(x,y)>0 \quad d(x,x)=0 
   $

   b. 对称性
   $
    \quad d(y,x) = d(x,y) \quad, x,y\in X
   $

    c. 三角不等式
