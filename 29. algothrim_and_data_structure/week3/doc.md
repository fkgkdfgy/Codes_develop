1. 单链表 
   带头节点的单链表
   头指针-> 头节点 -> .... -> 尾节点
   头节点没有存储任何信息

2. 单链表初始化：
   a. 生成头节点，并让头指针L指向头节点
   b. 头节点指针域置空

3. 判断 单链表是不是空的：
   头指针和头节点在，判断头节点的指针域是不是空的就可以

4. 销毁 Destroy：
   删除所有的节点：包括头节点

5. 清空 ClearList：
   删除除了所有头节点以外的节点

6. 时间效率分析：
   查找： O(n)
   插入：删除： 好的时候O(1) 不行的时候就是O(n)
